# Ралли Карелия

Одностраничный рекламный сайт для мероприятия "Ралли Карелия".

## Структура проекта

```
rally/
├── index.html          # Главная страница
├── video.mp4          # Видео для главного раздела
├── 1.png              # Изображение для баннера новостей
├── 2.png              # Изображение для первой новости
├── 3.png              # Изображение для второй новости
└── README.md          # Этот файл
```

## Особенности

- Адаптивный дизайн
- Автопроигрывание видео
- Интерактивные элементы с эффектами hover
- Кастомные шрифты (Suzuki Alstare Extreme Racing, MTS Wide)

## Развертывание на GitHub Pages

### Способ 1: Через настройки репозитория

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта в репозиторий
3. Перейдите в **Settings** → **Pages**
4. В разделе **Source** выберите ветку `main` (или `master`)
5. Нажмите **Save**
6. Ваш сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория/`

### Способ 2: Через GitHub CLI

```bash
# Инициализация git репозитория
git init
git add .
git commit -m "Initial commit"

# Создание репозитория на GitHub и загрузка файлов
gh repo create rally-karelia --public --source=. --remote=origin
git push -u origin main
```

После этого включите GitHub Pages в настройках репозитория.

## Важные замечания

⚠️ **Шрифты**: Для корректной работы сайта необходимо добавить файлы шрифтов:
- `SuzukiAlstareExtremeRacing.woff2` (или `.woff`, `.ttf`)
- `MTSWide.woff2` (или `.woff`, `.ttf`)

Если файлы шрифтов отсутствуют, сайт будет использовать резервные шрифты (Arial).
